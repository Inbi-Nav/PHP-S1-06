#  Práctica: PHP Avanzado — Formularios, POO y Composer  

**Descripción:**  
Esta práctica está diseñada para seguir profundizando en las características avanzadas del lenguaje **PHP**, combinando **formularios, programación orientada a objetos, enumeraciones, traits** y el uso del **gestor de dependencias Composer**.  
Los ejercicios están divididos en **tres niveles**, desde conceptos básicos hasta la integración de librerías externas.

---

- **Lenguaje:** PHP  
- **Entorno:** XAMPP o Laragon
